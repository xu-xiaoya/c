
C语言入门的10个步骤

1、下载群里的Dev-C++，安装。安装的时候会让你选中文。
2、打开Dev-C++，点文件 - 新建 - 源代码。
3、输入以下程序
#include<stdio.h>
int main() {
  printf("Hello,World!"); 
} 
4、点保存，文件名随意
5、点编译运行。祝贺你，运行出了第一个程序
6、再把中间的话改为任意文字、或者在文字中间加个\n试试。
7、输入或将前面的程序改为以下
#include<stdio.h>
int main() {
  int a, b, c;
  scanf("%d%d", &a, &b);
  c = a + b;
  printf("%d", c); 
} 
8、点编译运行，输入20 30，按回车
9、最后一行改为 printf("%d + %d = %d", a, b, c); 
10、下面自己改为减法或乘法或除法，再运行。然后继续祝贺，这是你自己编出的第一个程序

  int a, count = 0, sum = 0, max;
  double ave;
  scanf("%d", &a);
  max = a;
  while (a != -1) {
    sum += a;
    if (a > max) max = a;
    count++;
    scanf("%d", &a);
  }
  if (count != 0) {
    ave = (double)sum / count;
    printf("最大值：%d，平均值：%.2f\n", max, ave);
  }
}

示例34、牛顿迭代法求n的平方根

函数 f(x) = x^2 - n，求 f(x) = 0 的根。 
f'(x0) = f(x0) / (x0 - x1)   ==>   x1 = x0 - f(x0) / f'(x0)
代入f(x) = x^2 - n，得到 
牛顿迭代公式：
x1 =  (x0 + n/x0) / 2

#include<stdio.h>
double sqrt(int n) {
  double x0, x = n;
  do {
    x0 = x;
    x = (x0 + n / x0) / 2;
//算出的Xn和X（n+1）的差的绝对值小于某个值，即认为找到了精确的平方根。
  } while (x0 - x > 1e-16);
  return x;
}
int main() {
  int n;
  while (scanf("%d", &n), n)
    printf("%.15f\n", sqrt(n));	  
}
